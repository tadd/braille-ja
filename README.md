# Braille::Ja

日本語の点字を扱う変換器です

## Installation

Add this line to your application's Gemfile:

    gem 'braille-ja'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install braille-ja

## Usage

"あいうえお".to_tenji
